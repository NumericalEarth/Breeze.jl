# Validation Data

This directory holds local, non-CI reference datasets and helper environments
used to validate Breeze features during development.

Contents
- `validation/p3`: P3 kin1d reference data and reproduction notes.
- `validation/p3_env`: Local Julia environment used to write NetCDF files.
- `validation/topography`: Topography validation utilities.

Notes
- The files in `validation/` are intended for local use. Decide case-by-case
  whether to commit them.
